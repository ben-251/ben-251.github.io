What is Pll?

F2l stands for Permutation of the Last Layer. This is the last algorithm(s) of the solve and can be solved using 2-look or 1-look, where as the name suggests, you can either use one algorithm of 21, or stick with the 6 algorithms of beginner method

Tips

  • Aim to reduce regrips and rotations by inserting from the back as well as the front
  • Try to get efficient solutions for cases before developing muscle memory for them
